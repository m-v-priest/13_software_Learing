== Chrome 开发者工具

==== 查看请求头

开发者工具 -> Network  +
-> 左下角Name栏中, 随便选中一个请求的文件,  +
-> 在右侧Headers栏中, 找到最底部的Request Headers字段,  +
-> 里面就有 User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/76.0.3809.100 Safari/537.36
